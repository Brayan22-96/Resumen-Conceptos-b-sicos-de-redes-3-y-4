# Resumen-Conceptos-basicos-de-redes-3-y-4
## Módulo 3: Redes Inalámbricas y Móviles
Este módulo explica cómo los datos viajan a través del aire y cómo los dispositivos se mantienen conectados mientras se mueven.

Redes Inalámbricas (WLAN): Se basan en el estándar IEEE 802.11. A diferencia de las redes cableadas, utilizan ondas de radio para transmitir datos. Los componentes clave son los Puntos de Acceso (AP) y los adaptadores de red inalámbrica.

Conectividad Móvil: Explica la evolución de las redes celulares (4G/5G) y cómo permiten la comunicación de datos a gran escala.

Seguridad Inalámbrica: Es fundamental debido a que las señales viajan por el aire. Se mencionan métodos de cifrado como WPA2 y WPA3 para proteger la información contra accesos no autorizados.

## Módulo 4: Crear una Red Doméstica
En esta sección se aplica lo aprendido para entender cómo configurar y asegurar una red pequeña en el hogar.

Conceptos de Redes Domésticas: La mayoría de las redes del hogar utilizan un enrutador inalámbrico integrado, que combina las funciones de router, switch y punto de acceso en un solo dispositivo.

Tecnologías de Red: Uso de tecnologías como Ethernet para conexiones estables y Wi-Fi para movilidad. También se consideran las opciones de conectividad de banda ancha (DSL, Cable o Fibra).

Estándares Inalámbricos: Comparación de las bandas de frecuencia de 2.4 GHz (mayor alcance, pero más lenta y saturada) y 5 GHz (más rápida, pero con menor alcance).

Configuración del Enrutador: Pasos básicos para configurar el nombre de la red (SSID), la contraseña de seguridad y la actualización del firmware para mantener el equipo protegido.
